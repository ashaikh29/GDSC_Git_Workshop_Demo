# GDSC_Git_Workshop_Demo
Demo Repo for the GDSC Git Workshop

## Task
There are some bugs in `calculator.py`.

### Your job/objective
1. Fork this repo
2. Create a branch for bug fixing
3. Clone your forked repo
4. Checkout to the branch you created
5. Fix the bug/s
6. Commit your changes
7. Push to your repo
8. Open a pull request to the main repo
